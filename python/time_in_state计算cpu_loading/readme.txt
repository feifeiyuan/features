一、功能：计算cpu_loading数据


二、不同的需求不同情景使用的方法不同

1、连接USB一步操作获取loading

正确连接ADB之后，双击setup.bat脚本，出现如下界面时输入测试时长(默认测试60次)，enter开始执行
please input test times :
执行结束后consule界面，出现如下提示“请按任意键继续. . .”此时可通过按任意键退出
在setup.bat的同级目录下将生成对应的loading的结果文件result.txt


2、连接USB分步抓取数据和做数据处理

双击get_statistic.bat脚本，出现如下界面时输入测试时长(默认测试60次)，enter开始执行
please input test times :
执行结束后consule界面，出现如下提示“请按任意键继续. . .”此时可通过按任意键退出
在tool目录下将生成抓取到的数据文件

正确抓取数据之后，双击deal_statistic.bat，
执行结束后consule界面，出现如下提示“请按任意键继续. . .”此时可通过按任意键退出
此时将在同级目录下生成result.txt的同级目录文件

3、连接串口获取loading
将tool目录下的cpu_data.sh文件push到sdcard目录下，串口连接手机后
执行命令
sh sdcard/cpu_data.sh 测试时间（默认60s）
测试结束后将在/sdcard/result/生成对应的数据文件，
将该目录下的文件全部拷贝到tool目录下
然后双击deal_statistic.bat，执行结束将在同级目录下生成result.txt的同级目录文件
